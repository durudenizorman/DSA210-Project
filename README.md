# DSA210-Project
DSA210 Project

Data Enrichment: Integrated FDA FAERS API data with Drugs.com patient reviews to meet the enrichment requirement.

EDA: Extracted dosages from plain text reviews using string manipulation and visualized gender-based severity differences.

Hypothesis Tests: With correlation analyses, I found that while dosage alone does not linearly correlate with side effect severity or patient sentiment, gender is a statistically significant factor. Male patients report a higher average number of side effects. This finding will serve as a core feature for the Machine Learning model planned.
